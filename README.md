# 松风琴社 · 古琴官网

> 以现代设计语言诠释千年琴道

## 站点预览

直接用浏览器打开 `index.html` 即可查看。

## 项目结构

```
guqin-website/
├── index.html    # 单文件完整网站（HTML + CSS + JS）
└── README.md
```

## 设计风格

- **主题**：水墨禅意 × 现代奢华
- **字体**：思源宋体（正文） + 站酷小薇体（标题）
- **配色**：墨黑 `#1a1510` + 宣纸白 `#f5f0e8` + 古铜金 `#c9a84c`
- **动画**：入场渐显、滚动触发、平滑切换

## 页面结构

| 区块 | 内容 |
|------|------|
| Hero | 品牌主视觉 + 双 CTA |
| 理念 | 品牌故事 + 古琴文化介绍 |
| 藏品 | 三款古琴商品展示卡片 |
| 斫琴工艺 | 四步斫琴流程（选材/制形/髹漆/定音）|
| 琴学 | 古琴知识科普 |
| 知音评说 | 客户证言轮播 |
| 联系 CTA | 号召留联 |
| 页脚 | 导航 + 服务 + 联系方式 |

## 部署到 GitHub Pages

1. 创建新仓库，上传 `index.html`
2. 进入 **Settings → Pages**
3. Source 选择 `main` 分支和 `/(root)`
4. 等待 2 分钟，站点即上线

## 自定义

- 品牌名：`松风琴社`（全局搜索替换）
- 联系方式：搜索 `contact@` 替换为真实邮箱
- 商品信息：在 `#collection` 区块修改琴名、价格、描述
- 颜色变量：在 CSS `:root` 中修改 `--gold`、`--ink` 等值

## 技术说明

- 纯原生 HTML + CSS + JavaScript，无框架依赖
- 响应式设计，适配桌面 / 平板 / 手机
- Google Fonts CDN（需联网访问字体）
- 零外部图片依赖（古琴图形为 SVG 内联绘制）
